Возможности сайта

Публикация поста
Посты могут публиковать только авторизованные пользователи.
Изменение поста
Пост может изменить только пользователь, создавший пост.
Создание комментариев к посту
Комментарии могут оставлять только авторизованные пользователи.
Подписка на других пользователей
Подписка дает возможность просмотра постов авторов, на которых подписан пользователь.

Лайки постов
Пользователи могут оставлять лайки для понравившихся постов.
Количество просмотров постов
Увеличивается при просмотре поста целиком.

Группы
Возможность просмотров постов по группам.

Работа с пользователем

  • Регистрация пользователя;
  • Авторизация пользователя;
  • Смена пароля (для авторизоанных пользователей);
  • Сброс пароля через почту (для неавторизованных пользователей);
Личный кабинет для установки/изменения:
  • Личных данных (имя, фамилия, почта);
  • Аватара пользователя;
  • Персональных данных (дата родждения и др.).
